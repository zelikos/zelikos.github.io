About Me

I've been a passionate Linux user for over three years now; I've been a content creator (i.e. YouTuber) for more than twelve. In those three years of using Linux, my video content become a lot more focused on Linuxy things, as well as FOSS and tech in general. I wanted to create a central "brand" that all of my Linux, FOSS, and tech content could live under, so I went with AkzelTech.

This website serves as a hub for all of the content that I produce, and it is under constant development. You can follow the progress over on the GitHub page.

Head over to the Contact page if you want to get in touch directly (e.g. via Mastodon). If you'd like to support my work, check the Support page.
